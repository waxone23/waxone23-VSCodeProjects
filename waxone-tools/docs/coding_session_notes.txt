
6. TESTING TOOLS
----------------------
* touch [filename] : Create a new empty file.
* touch -t YYYYMMDDhhmm [filename] : Change a file's timestamp to the past (useful for testing age-checks).

7. CLEANING THE WORKSPACE
----------------------
* clear  : Wipes the screen text.
* Ctrl+L : Fast shortcut for 'clear'.
* reset  : Fixes the terminal if it starts acting glitchy.

8. EXIT CODE LEGEND
----------------------
* 0 : SUCCESS (Green Light).
* 1 : DATA ERROR (The input was bad/negative).
* 2 : USAGE ERROR (You forgot the argument).
* Shortcut: Use 'echo $?' to see the last exit code.
